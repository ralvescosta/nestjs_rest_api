## Installation

```bash
$ yarn
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov

$ yarn test --clearCache
```

## Ultils

### Migrations

```bash
$ yarn typeorm migration:create -n migrationNameHere

$ yarn typeorm migration:run

$ yarn typeorm migration:revert
```

### Seeds

```bash
$ yarn ts-node ./node_modules/typeorm-seeding/dist/cli.js seed
```
